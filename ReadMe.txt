Description: This application takes in two inputs from the user i.e. Number of products 
needed and the Budget of the user. The application then creates a list of Zappos products
whose combined values match the budget entered by the user.

The keys provided are not valid keys. To run the application insert working API keys in 
the file challengeJS.js

In order to run the application open the html file -- ZapposChallenge.html and enter the 
-desired inputs
